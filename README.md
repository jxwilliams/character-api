Character API
Overview

This project is a Spring Boot REST API that manages fictional characters.
The API allows users to create, read, update, delete, and search characters.

The application uses:

Spring Boot

Spring Data JPA

PostgreSQL

Maven

Postman for testing

Features

The API supports the following operations:

Create a new character

Retrieve all characters

Retrieve a character by ID

Update an existing character

Delete a character

Search characters by name

Filter characters by category


The application connects to a PostgreSQL database hosted on Neon.

Hibernate generates the table using the entity.
